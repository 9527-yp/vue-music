# 网易云音乐

**特别声明：本项目中的展示图片归网易云音乐所有。如果侵权，请提交 Issues 联系。** 

## 环境要求

需要 NodeJS 20.0.0+ 环境

## 简介

项目接口及接口文档请查看[NeteaseCloudMusicApi](https://github.com/Binaryify/NeteaseCloudMusicApi)

## 项目使用技术栈

* Vue
* pinia
* TypeScript

## 加入项目

如果想加入本项目开发，需将项目fork到自己的仓库下，功能开发完成后，提PR即可。

## 运行

启动项目

```
pnpm install
```

```
pnpm run dev
```

项目打包

```
pnpm build
```
